# DVDRental-SQL-Python-Practice-

I recently completed a Data Analytics bootcamp through UT Dallas/Fullstack Academy.
Early in 2026 while learning SQL, I took the DVDRental database we had been working with between 
classes to get some additional practice on my own.  My focus with this code was to look into slow 
moving/Dead inventory from a standpoint of what inventory should be replaced.  This was really 
the first time I took a dataset to SQL without a specific list of instructions to follow, it was
just, explore the data, see what you can find.  This was the first time I had fun with SQL as my
findings are genuinely interesting.

After finishing the bootcamp I decided to analyze the same database with Python as well.  I've 
really enjoyed working with Python these past weeks and wanted to compare how to explore the same
data with the same goal but with a different programming language.  I found it interesting how different
it was to do this with Python & Pandas compared to SQL, some of the findings, like the no inventory
films really stuck out and were a lot quicker to see in Python.  Then again when I was looking at the
data in Python I already knew what I was going to find, so that makes a difference as well.  I'm attaching
both files here.
